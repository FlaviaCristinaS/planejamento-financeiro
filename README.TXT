INSTRUCOES PARA EXECUTAR O PROJETO:

1) Acessar pasta 'build' -> digitar no terminal: make

2) O arquivo compilado 'planejamento-financeiro' sera gerado.

3) Na pasta 'build' existe um arquivo 'RUN.TXT' com um comando para ser executado no terminal.

4) Basta trocar os nomes dos arquivos de entrada e saida como preferir, copiar o comando
e colar no terminal. (lembre-se de estar no diretorio da pasta 'build')

5) Sera gerado um arquivo de relatorio um diretorio acima.

6) Fim.
